# Кирило Северов — Сайти під ключ + SEO

Односторінковий лендінг-портфоліо для послуг з розробки сайтів і SEO-просування.
Зроблено на чистих HTML / CSS / JS — без збірки, фреймворків і залежностей. Готовий до деплою на будь-який статичний хостинг.

## Структура

```
.
├── index.html              # головна сторінка (uk)
├── assets/
│   ├── styles.css          # дизайн-система та всі стилі
│   ├── main.js             # анімації, мобільне меню, валідація форми
│   ├── favicon.svg         # іконка
│   └── og.svg              # прев'ю для соцмереж (Open Graph)
├── robots.txt
├── sitemap.xml
└── site.webmanifest
```

## Дизайн

- Преміальна темна тема із золотим акцентом
- Шрифти: **Inter Tight** (заголовки) + **Inter** (текст) через Google Fonts
- Адаптив mobile-first, доступність (контраст ≥ 4.5:1, focus-стани, aria, клавіатурна навігація)
- Анімації появи з підтримкою `prefers-reduced-motion`
- Секції: Hero → Послуги → Процес → Портфоліо → Переваги → Тарифи → Гарантії → FAQ → Про мене → Контакти

## SEO «з коробки»

- Заголовок, опис, ключові слова, canonical
- Open Graph + Twitter Card
- Мікророзмітка JSON-LD: `ProfessionalService` + `FAQPage`
- `sitemap.xml`, `robots.txt`, семантична структура заголовків (один `h1`)

## ⚠️ Що замінити перед запуском (placeholder'и)

1. **Домен** — замінити `https://kirillseverov.site/` у `index.html` (canonical, OG, JSON-LD), `robots.txt`, `sitemap.xml`.
2. **Контакти** — Telegram `@kirillseverov`, email `hello@kirillseverov.site`, телефон `+380 (00) 000-00-00` у секції контактів і футері.
3. **OG-прев'ю** — за бажанням замінити `assets/og.svg` на JPG/PNG 1200×630 (краще індексуються в соцмережах) і оновити `og:image`.
4. **Фото** — у блоці «Про мене» зараз монограма-заглушка. Додайте реальне фото у `.author__photo`.
5. **Кейси** — у секції «Портфоліо» замінити заглушки на реальні скріншоти та цифри.
6. **Форма заявки** — зараз показує підтвердження без відправки. Підключіть бекенд:
   - простий варіант — [Formspree](https://formspree.io) / [Getform](https://getform.io): додати `action` і `method="POST"` у `<form id="lead-form">`;
   - або надсилати у Telegram через бота. Логіку сабміту дивіться у `assets/main.js`.

## Деплой

Підходить будь-який статичний хостинг — **Vercel**, Netlify, GitHub Pages, Cloudflare Pages.
Для Vercel: імпортувати репозиторій → фреймворк «Other» → деплой (build-команда не потрібна).
