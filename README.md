# Кирило Северов — Сайти під ключ + SEO

Односторінковий лендінг-портфоліо для послуг з розробки сайтів і SEO-просування.
Зроблено на чистих HTML / CSS / JS — без збірки, фреймворків і залежностей. Готовий до деплою на будь-який статичний хостинг.

## Структура

```
.
├── index.html              # головна сторінка (uk)
├── assets/
│   ├── styles.css          # дизайн-система та всі стилі
│   ├── main.js             # анімації, мобільне меню, валідація форми
│   ├── favicon.svg         # іконка
│   └── og.svg              # прев'ю для соцмереж (Open Graph)
├── robots.txt
├── sitemap.xml
└── site.webmanifest
```

## Дизайн

- Тепла «мокко» hero-секція з **яскраво-жовтим** акцентом (#FFD400), темні нижні секції
- Підпис-маркер (жовте виділення) на ключових словах, кнопки «cream pill + жовте коло зі стрілкою»
- Права інфо-панель у hero, секція порівняння **«Було → Стало»**, mono-HUD підписи
- Шрифти: **Inter Tight** (заголовки) + **Inter** (текст) + **JetBrains Mono** (технічні підписи) через Google Fonts
- Центрований портрет у hero — **плейсхолдер для фото Кирила** (`.hero__portrait .frame`)
- Адаптив mobile-first, доступність (контраст ≥ 4.5:1, focus-стани, aria, клавіатурна навігація)
- Анімації появи з підтримкою `prefers-reduced-motion`
- Секції: Hero → Послуги → Процес → Портфоліо → Переваги → Тарифи → Гарантії → FAQ → Про мене → Контакти

## SEO «з коробки»

- Заголовок, опис, ключові слова, canonical
- Open Graph + Twitter Card
- Мікророзмітка JSON-LD: `ProfessionalService` + `FAQPage`
- `sitemap.xml`, `robots.txt`, семантична структура заголовків (один `h1`)

## ⚠️ Що замінити перед запуском (placeholder'и)

1. **Домен** — замінити `https://kirillseverov.site/` у `index.html` (canonical, OG, JSON-LD), `robots.txt`, `sitemap.xml`.
2. **Контакти** — реальні: Telegram `@mr_devop`, телефон `073 666 18 36`, адреса `майдан Захисників України, 4`. Секція контактів — без форми: дві кнопки (Telegram / дзвінок) + плитки контактів.
3. **OG-прев'ю** — за бажанням замінити `assets/og.svg` на JPG/PNG 1200×630 (краще індексуються в соцмережах) і оновити `og:image`.
4. **Фото Кирила** — зараз підключене зовнішнім посиланням (ImgBB) у hero та блоці «Про мене». Для надійності рекомендую покласти фото локально: додайте файл **`assets/kirill.jpg`** і замініть у `index.html` обидва `src="https://i.ibb.co/..."` на `src="./assets/kirill.jpg"`. Якщо зображення не завантажиться — автоматично показується стилізований аватар `assets/avatar.svg`.
5. **Кейси** — у секції «Портфоліо» два реальні проєкти з прев'ю-макетами та посиланнями: [iCare](https://icare.com.ua/) (ремонт телефонів + AI-помічник) та [AutoKey](https://autokeykh.lovable.app/) (авто із США). Макети у `assets/cases/*.svg` можна замінити на реальні скріншоти.
6. **Зв'язок** — форму прибрано. Контакт лише через кнопки «Написати в Telegram» (`t.me/mr_devop`) та «Подзвонити» (`tel:073 666 18 36`).

## Деплой

Підходить будь-який статичний хостинг — **Vercel**, Netlify, GitHub Pages, Cloudflare Pages.
Для Vercel: імпортувати репозиторій → фреймворк «Other» → деплой (build-команда не потрібна).
